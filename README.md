# 理論株価アナリシス（東証プライム RIM モデル）

残余事業利益モデル（RIM: Residual Income Model）を用いた東証プライム銘柄の理論株価計算ツールです。

## 理論株価の計算式

```
理論株価 = ①正味営業資産/株 + ②正味金融資産/株 + ③予想残余事業利益の割引現在価値
```

- **①正味営業資産/株** = (営業資産 - 営業負債) ÷ 発行済株式数
- **②正味金融資産/株** = (現預金・有価証券 - 有利子負債) ÷ 発行済株式数
- **③残余事業利益PV** = Σ [(ROE_t - r) × BPS_t] ÷ (1+r)^t + 終端価値PV
  - r = 要求利回り（デフォルト5%、銘柄ごとに変更可能）
  - 予測期間: 1〜5年（ユーザー設定）
  - 終端価値: Gordon Growth Model

## データソース

Yahoo Finance API（`/v8/finance/chart`, `/v10/finance/quoteSummary`）
- 株価: 5分キャッシュ
- 財務データ: 1時間キャッシュ

## セットアップ

### ローカル開発

```bash
npm install
npm run dev
# → http://localhost:3000
```

### Vercel デプロイ（推奨）

```bash
# Vercel CLIをインストール
npm i -g vercel

# デプロイ（初回）
vercel

# 本番デプロイ
vercel --prod
```

GitHubリポジトリとVercelを連携すれば、pushするたびに自動デプロイされます。

## 機能

| 機能 | 説明 |
|------|------|
| リアルタイム株価 | Yahoo Finance APIから自動取得（5分更新） |
| 理論株価計算 | RIMモデルで全15銘柄を一括計算 |
| 予測期間設定 | 1〜5年をワンクリックで切り替え |
| 終端成長率調整 | スライダーで0〜5%を調整 |
| 要求利回り設定 | 銘柄ごとに個別設定（クリックして編集） |
| 年次明細表示 | 各年の残余事業利益・PVを表示 |
| 評価ランク | 乖離率に基づく5段階評価 |

## 銘柄追加

`src/lib/stocks.ts` の `PRIME_STOCKS` 配列に追記するだけです：

```typescript
{ code: "9999", yahoo: "9999.T", name: "追加銘柄", sector: "セクター名" },
```

## 注意事項

- 本ツールは教育・研究目的のシミュレーターです。投資判断の根拠としないでください。
- Yahoo Finance APIは非公式であり、仕様変更による取得失敗時はデモデータにフォールバックします。
- 予想ROEは今期実績から業界平均（8%）へ逓減する簡易推計です。実際のアナリスト予想とは異なります。
